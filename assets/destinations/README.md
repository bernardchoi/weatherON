# WeatherON Destination Images

목적지 케어 화면용 생성형 장소 분위기 이미지.

## 원칙

- 실제 장소 사진이나 특정 건축물을 복제하지 않음.
- 목적지의 환경·활동 성격만 반영한 가상 장면임.
- 이미지 내부에 로고, 상표, 간판, 읽을 수 있는 문구를 넣지 않음.
- 앱에서는 `AI 이미지`로 표시해 실제 현장 사진과 구분함.
- 기본 생성본은 Codex 내장 image generation 도구로 제작하고 모바일용 JPEG로 최적화함.

## 프롬프트 요약

- 안목해변: 한국 동해안 해변과 카페 거리의 차분한 아침 분위기.
- 잠실야구장: 서울 도심형 야구장의 경기 전 늦은 오후 분위기.
- 도쿄역: 붉은 벽돌과 현대 업무 지구가 만나는 대도시 철도 거점 분위기.
- 신사이바시역: 비 오는 저녁의 쇼핑 아케이드와 지하철 입구 분위기.
- 난바역: 블루아워의 복합 환승 거점과 보행 광장 분위기.
- Central Park: 도시 스카이라인을 배경으로 한 넓은 녹지의 늦은 오후 분위기.
- 공통 도시: 특정 지역을 식별할 수 없는 현대 보행 광장과 대중교통 분위기.

## 확장 프롬프트 방향

- 한국 교회: 구릉형 주거지, 석재 외관, 은행나무와 공동주택 환경.
- 일본 교회: 저층 주거지, 붉은 벽돌 예배당, 자전거와 기와지붕 환경.
- 유럽 교회: 석회암 교구 교회, 자갈 광장, 오래된 석조 주거지 환경.
- 한국 사찰: 단청 목조건축, 화강암 마당, 소나무와 산악 지형.
- 일본 신사·사찰: 목조 배전, 석등, 삼나무 숲과 젖은 자갈길.
- 한국 축구장: 공동주택과 낮은 산을 배경으로 한 도심형 전용 경기장.
- 일본 축구장: compact 관중석과 경량 지붕, 중층 도시와 녹지 환경.
- 유럽 축구장: 벽돌·철골 경기장이 오래된 주거지와 맞닿은 도심 환경.
- 한국 학교·병원·공항·산·문화시설: 공동주택, 구릉, 화강암, 소나무 등 지역 환경 반영.
- 일본 학교·병원·공항·산·문화시설: 저층 주거지, 정돈된 환승 구조, 삼나무·단풍, 목재와 콘크리트 환경 반영.
- 한국 철도·지하철·버스·항만: 대형 환승광장, 공동주택과 구릉, 화강암·유리·철골 기반 교통 환경.
- 일본 철도·지하철·버스·항만: compact 다층 환승, 자전거·타일 보행로, 저층 주거지와 정돈된 승강장 환경.
- 유럽 철도·지하철: 석조·벽돌 역전광장, 아치형 철골 지붕, 트램과 보행 공간이 결합된 환경.

모든 확장 애셋은 실제 시설 복제, 기관명, 간판, 로고, 국기, 스폰서 표기를 제외함.

## 매핑 예외 처리

- 신사이바시·도쿄역·난바는 검색 공급자가 다른 ID를 반환해도 이름과 주소 별칭으로 기존 전용 애셋을 우선 적용함.
- 한국어 `신사`는 독립된 단어일 때만 종교시설로 인식함. `신사이바시`, `신사동` 같은 지명 부분 문자열은 사찰·신사 이미지로 분류하지 않음.
