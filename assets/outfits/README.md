## Design Direction

- Use realistic item shapes, not abstract color blocks.
- Keep the item recognizable at small H1 thumbnail size and detailed enough for C1 cards.
- MVP uses generated product-cut PNGs.
- Production should replace these with user wardrobe photos or background-removed product images.
