# Changelog

## Naming decision — 2026-07-16

- 공식 디자인 시스템 이름을 Ambient Surface로 확정함.
- 시각 재료를 Matte Air, 핵심 원칙을 Soft Density · Quiet Signal · Text First로 고정함.
- Perfora Air, perfora-air.*, --pa-*, PA*는 v1.x 호환용 레거시 식별자로 유지함.

## 1.0.0 — 2026-07-10

- v0.1 토큰·컴포넌트·wireframe·대시보드·접근성 패치를 단일 안정 패키지로 통합함.
- 10개 컴포넌트와 5단계 state 모델을 stable로 고정함.
- 대비, 44px target, focus, reduced motion/transparency, forced colors 기준을 기본값으로 승격함.
- 로컬 schema, release manifest, build/check 명령을 추가함.
- Perfora Air를 당시 내부 코드명으로 제한하고 현행 WeatherON MVP와의 적용 경계를 명시함.
