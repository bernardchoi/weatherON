## 2. 스토어 문구

### 짧은 설명

날씨와 목적지에 맞춰 옷차림, 우산, 신발, 출발 준비를 알려주는 앱

### 긴 설명 초안

WeatherON은 오늘의 날씨와 목적지 상황을 함께 보고 외출 준비를 도와주는 날씨 기반 생활 앱입니다.

현재 위치의 날씨, 목적지 날씨, 강수 가능성, 바람, 체감온도, 일교차를 바탕으로 옷차림과 준비물을 추천합니다. 단순히 날씨만 보여주는 것이 아니라, 지금 나갈 때 무엇을 챙기면 좋은지 바로 확인할 수 있게 구성했습니다.

주요 기능:
- 현재 위치 기반 날씨 요약
- 날씨에 맞는 코디 추천
- 우산과 신발 준비 안내
- 목적지 검색 및 목적지 케어
- 출발 전 알림 조건 관리
- 계정 연결 후 목적지와 설정 저장
- 위치 권한 거부 시에도 수동 목적지 검색 가능

WeatherON은 한국 지역에서는 기상청 날씨와 Kakao Local 장소 검색을 우선 사용하고, 글로벌 목적지는 Open-Meteo와 Google Maps 계열 연동을 확장할 수 있는 구조로 설계되었습니다.

날씨 정보와 추천은 참고용입니다. 실제 외출, 이동, 안전 판단은 현장 상황과 공식 기상 안내를 함께 확인해 주세요.

| 항목 | 기준 | 준비 상태 |
|---|---|---|

| 대표 그래픽 | 1024x500 | `assets/store/android-feature-graphic-v1.png` 후보 생성 완료 |
| 휴대전화 스크린샷 | Android phone 최소 세트 | preview APK 후 캡처 |
| 7인치 태블릿 | Android 태블릿 지원 시 | 현재 미지원 |
| 10인치 태블릿 | Android 태블릿 지원 시 | 현재 미지원 |

권장 스크린샷 세트:
- H1 홈: 현재 날씨와 코디/우산 요약
- P1 목적지 검색: Kakao Local 결과
- G2 목적지 케어: 목적지 날씨/알림 상태
- C1 코디 추천: 날씨 기반 착장
- M/R 정책 허브: 개인정보/알림/광고 설정

대표 그래픽 방향:
- 제품 신호: WeatherON 로고, 날씨 기반 외출 준비, 목적지 케어
- 색상: Hybrid Chrome 기준 딥 네이비, 민트, 스카이 블루, 웜 골드
- 문구 후보: `날씨에 맞춰, 나가기 전에 ON`
- 현재 후보: `assets/store/android-feature-graphic-v1.png`
