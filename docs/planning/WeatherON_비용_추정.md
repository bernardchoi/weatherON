## 1-1. 신규 출시 로드맵 기준 비용 판단

| 출시 단계 | 비용 판단 | 비고 |
|---|---|---|
| MVP 0 | 추가 유료 API 없음 | 사용자 선택 이동수단과 거리 기반 fallback으로 출발시간 역산, 목적지 날씨 비교, 강수 타임라인 UI 검증 |
| MVP 1 | 무료 한도 내 가능성 높음 | Kakao Local, KMA, Open-Meteo 우선. Kakao Directions는 알림 신뢰성 검증에 필요할 때만 도입 |
| Beta/v1.0 | 푸시/서버 운영 비용 실측 필요 | 핵심은 알림 정시성, 오류/fallback, 서버 adapter 안정성. 광고/구독 비용 구조는 아직 제외 |
| v1.1+ | AI/여행/광고/구독 비용 재산정 | MVP 지표 확인 후 AI 장소 추천, 도보/해외여행, AdMob, 구독 결제 비용을 별도 산정 |

기획안 피드백 반영 후 비용 관리는 "전체 기능 동시 출시 비용"이 아니라 "핵심 알림과 목적지 판단 기능을 얼마나 안정적으로 운영할 수 있는가"에 맞춘다.

## 2. AI 모델 토큰 비용 — 기획서에 없어 별도 추정

기획서에는 AI 기능의 토큰 사용량·비용 데이터가 없어, 호출 빈도·토큰 크기를 가정해 자체 추정.

**AI 기반 기능 3개** (룰 기반 기본 코디·해외여행 체크리스트는 토큰 비용 없음, 제외):

| 기능 | 단계/등급 | 비고 |
|---|---|---|
| ① 홈 AI 추천 카드 | Phase 2, 무료/프리미엄 | 위치+날씨+시간대 → 장소 추천, 매일 호출 |
| ② AI 코디 개인화 (스타일 태그+7일 플래닝) | Phase 3, 프리미엄 | 룰 기반 코디 매트릭스 위에 Gemini 레이어 추가 |
| ③ F8 장거리 국토종주 AI 플래너 | 프리미엄 v2.0 | 최대 14일 구간별 분석 |

2026-06-29 기준으로 ①~③은 MVP 0~1 필수 범위가 아니다. AI 호출은 MVP 핵심 기능 재방문과 알림 가치가 확인된 뒤 v1.1+ 확장 후보로 다시 판단한다.

**가격 기준** (2026-06 검증, Gemini 2.5 Flash): 입력 $0.30 / 100만 토큰, 출력 $2.50 / 100만 토큰. 환율 ₩1,500/$ 가정.

### MAU 1,000 기준 월간 추정 (2026-06-20 계산)

| 기능 | 가정 (이용 비율 × 빈도) | 토큰(입/출, 회당) | 월 비용 |
|---|---|---|---|
| 홈 AI 추천 카드 | 100% × 1회/일 (30회/월) | 500 / 300 | $27.0 |
| AI 코디 개인화 | 10% × 4회/월 | 600 / 500 | $0.6 |
| 장거리 AI 플래너 | 5% × 1회/월 | 2,000 / 1,500 | $0.2 |
| **합계** | | | **약 $28/월 (약 ₩4.2만/월)** |

- 비용은 거의 전부 매일 호출되는 ①에서 발생. ②③은 합쳐도 $1 미만.
- MAU 1,000은 기획서 로드맵상 Phase 1~2 단계라 ②③(프리미엄/후순위)이 아직 출시 전일 수 있어, 실제 비용은 ①의 ~$27/월에 더 가까울 가능성.
- 실사용 빈도·전환율 데이터가 없는 가정 기반 추정이므로, 실측 데이터 확보 시 갱신 필요.

## 3. MAU 단계별 종합 비용 정리

| Phase / MAU | 비-AI API | AI(Gemini) 추정 | 비고 |
|---|---|---|---|
| Phase 1~2 (MAU ~1천~5천) | ₩0 | 약 ₩3~4만/월 (홈 AI 카드만) | 코디 고도화·플래너 미출시 가능 |
| Phase 3 (MAU ~1만, 여행 플래너 추가) | ₩0~₩1만 | 홈 카드 비례 증가 + ②③ 소액 추가 | MAU 10배 시 AI도 비례 ×10 추정 |
| Phase 4~ (MAU 3만↑) | 월 ₩5~15만 | MAU 비례 증가, 자릿수 확인 필요 | T-map 전환 시점과 겹침 |

> AI 비용은 호출량에 선형 비례 가정. MAU가 늘면 위 MAU 1,000 기준 단가 × (MAU/1,000)으로 단순 환산 가능하나, 실제로는 이용률·재방문 빈도가 달라질 수 있어 참고용.
