## 3. 알림 정시성 로그

알림 1건마다 기록한다. 앱 `알림 센터 > 최근 처리` 이력과 대조한다.

| 알림 종류 | 예정 시각 | 실제 수신 시각 | 지연(분) | 열람 여부 | 딥링크 도착 화면 | 비고 |
|---|---|---|---|---|---|---|
| 출발 알림 / 비 시작 / 비 그침 / 테스트 | | | | 열람/미열람 | H1/H4/H5/G2/M2 | |

판정 기준:

- 정시: 예정 시각 ±3분 이내
- 지연: +3분 초과 ~ +15분
- 실패: 미수신 또는 +15분 초과. 실패 시 당시 권한 상태와 배터리 최적화 설정을 함께 기록한다.
